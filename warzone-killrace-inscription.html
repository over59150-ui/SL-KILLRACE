Comment participer

Le tournoi est limité à 16 équipes. Suis les étapes dans l’ordre pour éviter les erreurs pendant la Kill Race.

1. InscriptionRemplis ton équipe au format Quatuor : 4 joueurs.
2. Code privéCopie et garde le code généré après validation.
3. Dépôt screensEntre ton code dans l’espace équipe puis envoie Top + Scoreboard.
4. ValidationUn admin valide les screens avant le classement.
5. Classement liveLe score et le Top Killers se mettent à jour après validation.
6. RemplacementPour changer un joueur, contacte un admin. Ne refais pas l’inscription.

Espace équipe - dépôt des screens

1. Entre ton codeUtilise le code reçu après l'inscription pour accéder à ton équipe.
2. Choisis la bonne mancheFais un dépôt séparé pour chaque game retenue. Exemple : 3 games retenues = 3 envois.
3. Ajoute les résultatsIndique le numéro de game, les kills par joueur et le placement de l'équipe.
4. Envoie les preuvesAjoute les 2 captures demandées pour cette game. Le classement sera mis à jour après validation des admins.
Ne partage pas ton code hors de ton équipe. Si ton code ne fonctionne pas, contacte un admin et ne refais pas une inscription.
